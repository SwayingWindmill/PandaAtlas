## Scope and outcome

The completed pipeline is:

1. reviewed source registry and request policy;
2. deterministic fixture or approved live HTTP acquisition;
3. evidence snapshots and field-level candidates;
4. conservative identity reconciliation and conflict classification;
5. append-only curator decisions;
6. provenance-preserving curation patch proposal export.

The map remained command-line and data-artifact only. It introduced no curator-facing browser surface and no cross-page frontend publication. Existing browser and accessibility suites were run only as release-regression checks.

## Reproducibility and acceptance matrix

The acquisition integration matrix covers:

- repeated unchanged fixtures retaining identical evidence snapshot IDs, candidate IDs, and deduplication keys even though run and bundle IDs change;
- changed source bytes producing a new evidence snapshot, candidate ID, and deduplication key while preserving the prior artifact;
- duplicate candidate refusal with an evidence-only failed terminal bundle;
- conflicting reviewed source keys and external identifiers retaining all competing identities as `ambiguous`;
- ambiguous identities being refused during patch export;
- contradictions retaining the prior trusted value and assertion IDs and being refused during patch export;
- HTTP 429 producing a blocked, evidence-only terminal bundle with `rate-limited` block state and `Retry-After` evidence;
- expired source review refusing patch export;
- bundle, decision-log, and patch models rejecting non-empty trusted, canonical-curation, or publication write targets.

## Rollback and withdrawal

Acquisition outputs are immutable local JSON artifacts. A changed source creates a new lineage rather than overwriting prior evidence. Curator decisions are append-only. Patch export can be withdrawn by discarding the local proposal because it has no canonical write target.

The repository release-recovery drill independently passed all immutable-release rollback and withdrawal scenarios. No staging or Production activation was performed for this acquisition map.
